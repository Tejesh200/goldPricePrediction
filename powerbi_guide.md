# 📊 Power BI Dashboard Setup Guide

This guide explains how to import the dataset generated by our Machine Learning pipeline into **Power BI Desktop** and create a professional, stunning dashboard.

## 📂 Step 1: Import the Data
The ML pipeline generated an optimized CSV specifically structured for Power BI analysis.
1. Open **Power BI Desktop**.
2. Click on **Get data** > **Text/CSV**.
3. Navigate to your project folder: `goldPricePrediction/data/processed/`.
4. Select `powerbi_export.csv` and click **Open**.
5. Power BI will detect columns like `Date`, `Actual_Close`, `Predicted_Next_Close`, `Prediction_Error_Percentage`. Click **Load**.
6. *(Optional)* Let's also import the Feature Importance metrics!
   - **Get data** > **Text/CSV**
   - Navigate to `goldPricePrediction/models/` and select `feature_importance.csv`. Load it.

## 🎨 Step 2: Styling the Dashboard (Premium Look)
To match the aesthetics of the Streamlit App:
1. Go to the **View** tab in the Ribbon.
2. Under **Themes**, click the dropdown and select **Customize current theme**.
3. Change the Background to a dark color (e.g., `#0D1117`) and Text color to light gray (`#C9D1D9`).
4. Click **Apply**.

## 📈 Step 3: Build the Visualizations

### 1. KPI Cards (Top Row)
1. Add a **Card** visual.
2. Drag `Prediction_Error_Percentage` to the Fields pane. Ensure aggregation is set to **Average**.
3. Format the Card to add a title like **"Avg Prediction Error"**.
4. Add another **Card** visual. Drag `Direction_Correct` and set aggregation to **Average**. Change format type to Percentage. Title it **"Directional Accuracy"**.

### 2. Time Series Chart (Center Piece)
1. Add a **Line chart** visual and make it large to dominate the dashboard.
2. Drag `Date` to the **X-axis**. (Make sure it's set to "Date" rather than "Date Hierarchy" to show continuous flow).
3. Drag `Actual_Next_Close` to the **Y-axis**.
4. Drag `Predicted_Next_Close` to the **Y-axis** (it will plot below Actual).
5. Open the **Format your visual** pane (the paint roller icon):
   - Under Lines, set the Color for Actual to **Cyan** (`#00FFCC`) and Predicted to **Neon Pink** (`#FF007F`).
   - Change the `Stroke Style` for Predicted to **Dashed**.

### 3. Feature Importance (Side/Bottom Plot)
1. Add a **Clustered bar chart**.
2. Drag `Feature` (from the `feature_importance` dataset) to the **Y-axis**.
3. Drag `Importance` to the **X-axis**.
4. Sort the visual by Importance (Descending) to see what drives gold prices.

### 4. Slicers
1. Add a **Slicer** visual.
2. Drag `Date` into the field. This allows users to filter the dashboard timeframe easily.

## 🎉 Conclusion
Your Gold Price Prediction Power BI Dashboard is now complete! It dynamically links to the CSV file generated by your Machine Learning model. Anytime you update the data via `python src/predict.py`, merely hit **Refresh** in Power BI to fetch the latest predictions!
